CSS Cheat Sheet

Selectors: Choosing What to Style

selector applies to
b { all bolded content
i { all italisized content
h1, h2, ..., h6 { all headings of each respective weight
anyElement1, anyElement2, anyElement3 { all content using any of the elements
.className { all elements that have a class of className
elementName.className { all elementName's within className's
elementName1 elementName2 { all elementName2 content inside of elementName1
elementName1, elementName2 { all elementName2 content which has a parent of elementName1
elementName1 + elementName2 { the first elementName2 content that is placed immediately after elementName1
elementName1 ~ elementName2 { every elementName2 content that is preceded by an elementName1
[attributeName] { all elements with attributeName
[attributeName=attributeValue] { all elements with attributeName and attributeValue (i.e. [target=_blank])
[attributeName*=attributeValue] { all elements with attributeName and value that contains attributeValue (i.e. a[href*="w3schools"])

Styling Text

element description
b: changes content to bold
i: changes content to italics
h1, h2, ..., h6: changes content to the weight of each respective heading
font-family: changes font family
font-size: changes font size
font-weight: changes font weight

Positioning Content

element description
align-content: alings content (to left, right, center)
background-color/image/size/position: changes aspects of background
border-width/style/color: changes aspects of each respective border
border-top/bottom/left/right: changes placement of each respective border (solid red, dotted blue, double, etc)
margin-top/bottom/left/right: changes margins of each respective part of page
max/min-height/width: changes minimum or maximum space content can take up
padding-top/bottom/left/right: changes padding on an piece of content
